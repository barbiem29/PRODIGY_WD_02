# Stopwatch Web Application

This is a simple and interactive stopwatch web application built using HTML, CSS, and JavaScript. The stopwatch allows users to start, pause, and reset time intervals with a clean user interface.

## Features

- Start, pause, and reset functionality
- Time displayed in HH:MM:SS format
- Fully responsive and centered layout
- Smooth UI with basic animations
- Built using only HTML, CSS, and JavaScript (no frameworks)

## Technologies Used

- HTML5 for structure
- CSS3 for styling and layout
- JavaScript (vanilla) for interactivity and timing logic
